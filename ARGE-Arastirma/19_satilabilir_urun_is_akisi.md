# 19 - Satılabilir Ürün Belirleme İş Akışı (10 Adım)

---

## HEDEF

```
Dün trend + bugün trend + dün satan + bugün satan
→ ortak modeli bul → satılabilir ürün üret
```

---

## PLATFORM KURALI (DEĞİŞMEZ)

| Platform | Rol | NE VERİR |
|---|---|---|
| **TikTok** | Erken sinyal | Ne yükseliyor |
| **Instagram** | Erken sinyal + stil | Trend doğrulama |
| **Pinterest** | Tasarım yönü | Stil/renk/form |
| **Trendyol** | **Gerçek satış** | Satış doğrulama |
| **Google Trends** | Destek | Arama hacmi |

> ⚠️ Platformlar karıştırılmaz. Satış kararı **sadece** Trendyol'dan çıkar.

---

## 10 ADIM

### ADIM 1 — TREND TESPİTİ
- **Kaynak:** TikTok / Instagram
- **Bak:** İzlenme artışı, beğeni artışı, içerik çoğalması
- **Yöntem:** Aynı içerik 2–3 zaman screenshot → artış var mı?
- ✔ Artış varsa → trend adayı

### ADIM 2 — SATIŞ DOĞRULAMA
- **Kaynak:** Trendyol
- **Bak:** Yorum artıyor mu? Stok azalıyor mu?
- **Yöntem:** Ürün sayfası screenshot
- ✔ Satılıyorsa → gerçek aday
- ❌ Satış yoksa → RED

### ADIM 3 — ZAMAN ANALİZİ
- **Veri:** T-3 / T-1 / bugün karşılaştırma
- **Bak:** Sürekli artış mı, ani sıçrama mı?
- ✔ Sürekli artış → güçlü trend
- ❌ Ani artış + düşüş → balon

### ADIM 4 — BALON FİLTRE
- ❌ Sadece sosyalde var → RED
- ❌ Sadece influencer ürünü → RED
- ❌ Sadece reklam ürünü → RED
- ✔ Trend + satış birlikte → devam

### ADIM 5 — MODEL PARÇALAMA
Her ürün için ayrı ayrı çıkar:
- Kalıp (oversize/dar/regular)
- Renk
- Boy/uzunluk
- Form/siluet
- Kesim detayı

### ADIM 6 — ORTAK MODEL BULMA
```
Dün trend + bugün trend → ortak özellikler
Dün satan + bugün satan → ortak özellikler
Bu iki ortak birleşince → üretilecek model
```

### ADIM 7 — DAVRANIŞ ANALİZİ
- **Kaynak:** Yorumlar
- **Bak:** Neden alıyor? Neden almıyor?
- Şikayet eden neye şikayet ediyor?
- "Aldım, memnunum" diyen ne diyor?

### ADIM 8 — FIRSAT BULMA
- Eksik beden var mı?
- Eksik renk var mı?
- Kalite açığı var mı?
- Daha iyi fiyat yapılabilir mi?

### ADIM 9 — TEST (EN KRİTİK)
- 24 saat sonra tekrar kontrol
- 72 saat sonra tekrar kontrol
- ✔ Devam ediyorsa → güçlü
- ❌ Düştüyse → RED

### ADIM 10 — SON KARAR

| Şart | Durum |
|---|---|
| Trend var | ✔ |
| Satış var | ✔ |
| Zaman devam ediyor | ✔ |
| Test geçti | ✔ |
| **Karar** | **ÜRET** |

---

## ANA KARAR FORMÜLÜ

```
Trend (artış) + Satış (kanıt) + Zaman (süreklilik) + Fırsat (boşluk) = DOĞRU ÜRÜN
```

---

## RED KRİTERLERİ (Hata Engeli)

| Durum | Karar |
|---|---|
| Sadece sosyal, satış yok | ❌ RED |
| Sadece influencer ürünü | ❌ RED |
| Sadece reklam ürünü | ❌ RED |
| Ani düşüş | ❌ RED |
| Yorum yok | ❌ RED |

## %100'E YAKLAŞTIRAN EK KONTROLLER

| Kontrol | Soru |
|---|---|
| 3 platform kontrolü | Aynı ürün TikTok + IG + Trendyol'da var mı? |
| Farklı hesap kontrolü | Farklı hesaplar paylaşıyor mu? |
| Satın alma kontrolü | Yorumda "aldım" diyen var mı? |
| Süreklilik kontrolü | 3 gün devam etti mi? |
